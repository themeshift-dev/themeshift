# SplitPaneShell

For layouts where two regions matter at the same time.

## Use it for:

- auth pages with branding panel + form panel
- docs with nav + article
- mail/message list + detail
- gallery + preview
- editor + live preview

## Typical structure:

- left/right or start/end panes
- resizable or fixed ratios
- pane collapse behavior
- clear main-region ownership

This is less about “sidebar” and more about “dual-primary regions.”
