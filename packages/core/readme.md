Framework-agnostic token build and watch engine for ThemeShift.

## API

```ts
import { buildTokens, defineConfig, watchTokens } from '@themeshift/core';
```

- `defineConfig(config)`
- `buildTokens(options?)`
- `watchTokens(options?)`
- `auditTokens(options?)`
